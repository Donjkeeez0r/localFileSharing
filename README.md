____
## Простой файловый сервер на чистом Node.js для загрузки, скачивания и удаления файлов через веб-интерфейс.
____
### Сервер запускается на локальном IP
### Функционал:
- Загрузка файлов через форму
- Список всех загруженных файлов
- Скачивание файлов
- Удаление файлов

project/
├── index.js          # Основной сервер
├── public/
│   └── index.html    # Веб-интерфейс
└── public/uploads/   # Загруженные файлы (создается автоматически)
